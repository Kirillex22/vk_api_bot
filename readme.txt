How to use this:

1) go to "data" path

2) paste your targets into "targets.json" in format "name" : "vk_id"

3) go to ".env" file

3) paste your vk api token: to do this visit https://vkhost.github.io/, click at "vk.com", agree and copy all the text after 
   "access_tocken =..." and before "&expires" from the address line

4) paste your vk_id

5) execute app.exe and choose your target -> click start, you can loop this process; if you need to stop bot just click stop, but 
   this action will remove bot from all the dialogs 

6) i'm lazy and don't wanna describe other features you can see :)
